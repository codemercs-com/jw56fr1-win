Dieses Beispiel nutzt die standard Joystick Schnittstelle des Systems. Mit dieser Software
kann man alle Einstellungen am JW56FR1 vornehmen und darstellen.
ACHTUNG: Bei Start des Programms werden automatisch 2G, 245 DPS und keine Filter (Werkseinstellungen) eingestellt.

This example uses the standard joystick interface of the system. With this software
you can make and display all settings on the JW56FR1.
ATTENTION: When starting the program 2G, 245 DPS and no filters (factory settings) are set automatically.
